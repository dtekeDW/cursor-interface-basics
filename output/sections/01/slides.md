# Section 01 — Agent Tools

Tools are the building blocks of Agent.

- semantic search
- search files/folders
- web
- fetch rules
- read files
- edit files
- run shell commands
- browser
- image generation

Docs: [Agent overview](https://cursor.com/docs/agent/overview)

---

# Why this matters

- one task can combine many tools end-to-end
- no strict hard limit on tool calls per task
- result: less manual context switching in day-to-day work
