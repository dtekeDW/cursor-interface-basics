## Workflow

1. Update **[workshop.md](../../workshop.md)** first (agenda, minutes, doc links, beats).
2. Sync **[00-roadmap.md](00-roadmap.md)** status if scope changes.
3. Update affected **[output/sections/*/presentation.md](../../output/sections/)** and **[slides.md](../../output/sections/)** for the same block.
4. Every section file keeps navigation: **Workshop**, **Prev**, **Next**.
5. Keep time limits: **~30 minutes** talk + **10 minutes** Q&A.
6. Bump **version** and date in `workshop.md`.
7. Append a short line to **[output/changelog.md](../../output/changelog.md)**.
